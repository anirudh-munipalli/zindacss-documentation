Custom values in ZindaCSS

Custom Values

You can add custom values in Zinda CSS. In case a certain value is not predefined, you can make it yourself.
Example:

<div class="color_#ffeedd">Test</div>
<div class="w_500px">Test</div>
<div class="pad_3%">Test</div>
<div class="shadow_3px">Test</div>
It has to be separated by underscore(_);

You can have the following values for the property:

  • color_ [This should be used for color]
  • bg_ [This should be used for background color]
  • border_ [This is to specify border color]
  • w_ [This is used to specify width]
  • h_ [This is used to specify height]
  • pad_ [For padding]
  • pad-left_ [For padding on left]
  • pad-right_ [For padding on right]
  • pad-top_ [For padding on top]
  • pad-bottom_ [For padding on bottom]
  • margin-left_ [For margin]
  • margin-right_ [For margin on right]
  • margin-top_ [For margin on top]
  • margin-bottom_ [For margin on bottom]
  • shadow_ [For blur shadows]

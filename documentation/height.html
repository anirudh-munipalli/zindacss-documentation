Height in ZindaCSS

You can set the height of an object using a value from one to ten.

Predefined values

This is a random test object

Custom values

You can also have custom values height the following syntax.
<div class="height_value">Random object</div>
Examples:
<div class="height_10vh">Random object</div>
<div class="height_15%">Random object</div>
<div class="height_3em">Random object</div>
<div class="height_6.5rem">Random object</div>

Max Height

You can also set max-height values using max-height-Number
